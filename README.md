# Personal Portfolio — Chinta Lakshmi Nivas Jaswanth

A minimal, premium personal portfolio built with React, TypeScript, Tailwind CSS, and Framer Motion. Engineered for performance, accessibility, and clean typography, drawing inspiration from Linear and Vercel.

## 🛠️ Tech Stack

- **Framework:** React 18 (Vite)
- **Language:** TypeScript
- **Styling:** Tailwind CSS (Custom theme scale)
- **Motion:** Framer Motion (Restrained animations)
- **Icons:** Lucide React
- **SEO & Meta:** Full Open Graph tags, descriptive titles, accessible structured markup.

## 🚀 Getting Started

### Prerequisites

- Node.js (v18.x or later)
- npm or yarn

### Installation

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the local development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## 📐 Design & Structure

- **Dark Theme by Default:** Grounded around `#0A0A0B` (near-black) and `#0F0F11` elevation surfaces.
- **Accents:** Electric Blue (`#2B84E8`) utilized strictly for key focus indicators, CTAs, and active navigation links.
- **Content-Driven:** Zero filler copy or placeholder paragraphs. All copy uses structured data definitions in `src/data/`.
- **Performance:** Optimized bundle sizing and clean modular code.
